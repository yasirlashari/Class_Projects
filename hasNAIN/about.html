What is this?

This competition allows students to showcase their creativity and skills in website design and frontend development. Participants will design responsive and interactive websites using modern techniques.

Requirements

  • Use HTML & CSS
  • Apply Flexbox and CSS Grid
  • Include transitions & animations
  • Responsive design using media queries

Why Participate?

By joining, students gain practical experience, enhance their portfolio, and compete for recognition and prizes in a professional web development challenge.
